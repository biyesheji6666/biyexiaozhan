# 毕业设计辅导服务 - 演示文稿

这是一个完整的毕业设计辅导服务策划方案演示系统，包含客户版和内部汇报版两个HTML演示文稿。

## 📋 项目简介

本项目为计算机专业大四本科生提供毕业设计辅导服务的完整策划方案，包括：
- 💼 **内部汇报版**：面向公司管理层的商业计划书
- 🎓 **客户版**：面向学生客户的服务介绍

## 🌐 在线访问

## 🚀 GitHub Pages 部署步骤

### 方法一：通过网页操作（推荐新手）

1. **Fork或上传文件到GitHub**
   - 登录 [GitHub](https://github.com)
   - 点击右上角 "+" → "New repository"
   - 仓库名称：`graduation-project-service`（或其他名称）
   - 选择 "Public"
   - 点击 "Create repository"

2. **上传文件**
   - 在新建的仓库页面，点击 "uploading an existing file"
   - 将以下文件拖拽上传：
     - `index.html`（首页导航）
     - `客户版-毕业设计辅导服务介绍.html`（桌面版客户演示）
     - `mobile-client.html`（手机版客户演示）
     - `内部汇报-毕业设计辅导项目.html`（桌面版内部汇报）
     - `mobile-internal.html`（手机版内部汇报）
     - `client-presentation.html`（客户版英文跳转）
     - `internal-presentation.html`（内部版英文跳转）
     - `README.md`（项目说明）
     - `.gitignore`（Git配置）
     - `毕业设计辅导服务策划方案.md`（完整策划）
   - 点击 "Commit changes"

3. **启用GitHub Pages**
   - 进入仓库的 "Settings" → "Pages"
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main" 或 "master"，文件夹选择 "/ (root)"
   - 点击 "Save"

4. **等待部署完成**（约1-2分钟）
   - 页面会显示：`Your site is live at https://你的用户名.github.io/仓库名/`
   - 点击链接即可访问

### 方法二：使用Git命令（推荐开发者）

```bash
# 1. 克隆或初始化仓库
git init
git add .
git commit -m "Initial commit: 毕业设计辅导服务演示文稿"

# 2. 关联远程仓库
git remote add origin https://github.com/你的用户名/仓库名.git

# 3. 推送到GitHub
git branch -M main
git push -u origin main

# 4. 在GitHub仓库设置中启用Pages（见方法一步骤3）
```

## 💡 功能特点

### 桌面版（翻页式演示）
- ✅ **PPT式体验**：左右翻页，适合演讲展示
- ✅ **键盘控制**：方向键、空格键翻页
- ✅ **全屏模式**：F11全屏演示
- ✅ **动画效果**：平滑的页面切换动画
- ✅ **触摸支持**：平板设备左右滑动

### 手机版（垂直滚动）
- ✅ **触摸优化**：专为移动设备设计
- ✅ **垂直滚动**：符合手机浏览习惯
- ✅ **大字体易读**：适合小屏幕阅读
- ✅ **快速导航**：一键返回顶部
- ✅ **快速加载**：优化的移动端性能

### 共同特点
- ✅ **纯静态页面**：无需服务器，完全免费
- ✅ **自动设备检测**：智能识别设备类型
- ✅ **专业美观**：精心设计的配色和排版
- ✅ **无依赖**：不需要任何外部库
